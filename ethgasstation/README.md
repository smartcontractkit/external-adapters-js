# Chainlink External Adapter for EthGasStation

## Input Params

- `speed`: The speed for gas price to get. Available choices:
  - `safeLow`
  - `average` (default)
  - `fast`
  - `fastest`
- `endpoint`: The endpoint to use (optional, default: ethgasAPI)
