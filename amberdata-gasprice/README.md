# Chainlink External Adapter for Amberdata gas price

## Input Params

- `speed`: The speed for gas price to get. Available choices:
  - `safeLow`
  - `average` (default)
  - `fast`
  - `fastest`
- `endpoint`: The blockchain id to get gas prices from (optional, default: ethereum-mainnet)
