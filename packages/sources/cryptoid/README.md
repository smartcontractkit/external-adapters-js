# Chainlink External Adapter for cryptoID

## Input Params

- `blockchain` or `coin`: The blockchain name (required).
- `endpoint`: The requested data point. One of (`difficulty`|`height`). Defaults: `difficulty`.

## Output

```json
{
  "jobRunID": "1",
  "data": {
    "result": 19298087186262.6
  },
  "result": 19298087186262.6,
  "statusCode": 200
}
```
