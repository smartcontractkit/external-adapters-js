### Example

Request:

```json
{
  "id": "1",
  "data": {
    "signature": "function symbol() view returns (string)",
    "address": "0xdAC17F958D2ee523a2206206994597C13D831ec7",
    "endpoint": "function"

    "cacheKey": "i+gWrJZw68WU1SlaR+u5dJb+fbA="

}
```
