## Forex Endpoint

## Singular Currency Pair Example

## Input

```json
{
  "id": "1",
  "data": {
    "base": "USD",
    "quote": "EUR"
  }
}
```

## Outpput

```json
{
  "jobRunID": "1",
  "result": 0.841406,
  "debug": {
    "staleness": 0,
    "performance": 0.224847981,
    "providerCost": 1,
    "batchablePropertyPath": ["base", "quote"]
  },
  "statusCode": 200,
  "data": {
    "result": 0.841406
  }
}
```
