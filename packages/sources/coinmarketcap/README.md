# Chainlink CoinMarketCap Pro External Adapter

| Required? |  Name   |                            Description                             | Options | Defaults to |
| :-------: | :-----: | :----------------------------------------------------------------: | :-----: | :---------: |
|    âœ…     | API_KEY | An API key that can be obtained from [here](https://coinmarketcap.com/api/) |         |             |

---

### Input Parameters

| Required? |   Name   |     Description     |                                                    Options                                                     | Defaults to |
| :-------: | :------: | :-----------------: | :------------------------------------------------------------------------------------------------------------: | :---------: |
